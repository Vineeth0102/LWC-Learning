<h2>CSS</h2>
    <b>CSS</b> stands for <b>Cascadin Style Sheets.</b><br>
    CSS is used to control the style of a web page.

<h2>CSS Syntax</h2>
    selector <b>{property: value}</b><br>
    <b>selector</b> - it is used to "find" the HTML element.<br>
    <b>property</b> - CSS property you want to apply like color, border etc.<br>
    <b>value</b> - it's the value assigned to property like for color property value is red.<br>

<h2>CSS Selectors</h2>
    CSS selector are used to "find" (or select) the HTML elements you want to style.<br>
    <b>Element selector</b> - use of HTML tags for styling.

    e.g: p{color:blue;}

<b>ID selector</b> - uses the id attributeof an HTML element to select a specific element. To select an element with a specific id, write a hash(#) character, followed by the id of the element. The id of an element should be unique within a page.

    e.g: #Username{color:red;}

<b>Class Selector</b> - uses the class attribute of an HTML element to select a specific element. To select an element with a specific class, write a dot(.) character, followed by the class of the element.

    e.g: .Username{color:blue;}

<h2>Type of CSS styles</h2>
There are Three types of CSS styles
<ol>
    <li>Internal CSS - Not used in LWC</li>
    <li>Inline CSS</li>
    <li>External CSS or Third party library - Bootstrap, salesforce lightning design system</li>
</ol>
