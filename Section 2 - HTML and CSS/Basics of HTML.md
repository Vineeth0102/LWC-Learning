HTML - Hyper Text Markup Language

    Hyper Text refers to the way in which Web Pages(HTML documents) are linked together
    Markup Language is a computer Language that uses tags to define elements within a document.
    HTML Describes the structure of Web Page.
    HTML elements tell the browser how to display the content. 

Most commonly used HTML elements in LWC:
    An HTML elemen is defined by a start tag, some content and an end tag:
        <tagName> content goes here ... </tagName> 
        e.g <h1> My First Heading </h1>

    <div> - This tage defines a division in an HTML document.
    <p> - This tag used for paragraph that starts on a new line and some margin before and after a paragraph.
    <h1> -- <h6> - These tags are used to give Headingof diffrent size.
    <a> - This tag defines a hyperlink
    <img> - This tags help you to insert an image in web page.
    <ul> - This will list items using plain bullets.

HTML Attributes;

    HTML attributes provide additional information about HTML elements.
    Attributes are always specifed in the start tags
    Attributes usually come in name/value pairs like: name="value"
    e.g: <a href="https://google.com">Google</a>

HTML data-* Attribute

    The data-* attribute is used to store custom data private to the page or application
    e.g: <div data-name="vineeth">Username</div>

HTML Block and Inline elements
    
    Block level - Elements always starts on a new line and takes up the full width available
    e.g <div>, <p>, <h1>-<h6>, <ul>, <header>, <article>

    Inline Level - Element does not start on a new line and it only takes up as much width as necessary
    e.g <span>, <img>, <a>, <label>, <strong>
